# Ejercicios de la asignatura SAD (Sistemas de Apoyo a la Decisión)

Este es un repositorio en el que contribuir con los ejercicios que hemos conseguido que funcionen. Un esfuerzo conjunto para aprobar la asignatura, contribuye lo que puedas.

Se pide consistencia el el árbol de ficheros. La pruebas de implementación van en la carpeta examenesPracticos, cada una en su propio paquete examenX. Las prácticas de laboratorio van en la caprtea practicas, cada una en su propio paquete practicaX.

Cada práctica/examen debe ser independiente y estar contenido dentro de su paquete (aunque eso cause duplicar la misma clase en distintas prácticas/exámenes) para facilitar el exportar las prácticas/exámenes individualmente. Lá única excepción es la carpeta src/files, a la que siempre se puede acceder aunque esté fuera del paquete de la práctica/examen.
